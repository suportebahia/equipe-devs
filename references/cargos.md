# Competências por Cargo

## Liderança

### CTO (Diretor de Tecnologia)
- Visão de arquitetura de longo prazo
- Inovação e P&D
- Alinhamento tech-negócio
- Prospecção de tendências

### CPO (Diretor de Produto)
- Visão de portfólio
- Priorização de investimentos
- Descoberta de oportunidades
- Valor de mercado

### Gerente de Engenharia / Delivery Manager
- Gestão de pessoas
- Mentorias
- Facilitação ágil
- Previsibilidade de entregas

## Engenharia

### Arquiteto de Software
- Desenho de soluções escaláveis
- Segurança
- Integração entre sistemas
- Definição de padrões técnicos
- Escolha de stack tecnológica

### Desenvolvedor Front-end
- HTML, CSS, JavaScript/TypeScript
- React, Vue.js, Angular
- Performance de renderização
- Acessibilidade WCAG
- Design responsivo
- Consumo de APIs

### Desenvolvedor Back-end
- Node.js, Python, Java, C#, Go, PHP
- APIs REST/GraphQL
- Lógica de negócio
- Padrões de projeto
- Segurança server-side

### Desenvolvedor Full-Stack
- Visão integrada sistema
- Front e back-end
- Prototipagem rápida

### DBA (Especialista Banco de Dados)
- Modelagem de dados
- SQL e NoSQL
- Tuning de queries
- Backup e replicação
- Alta disponibilidade
- Recuperação de desastres

### DevOps / SRE
- IaC (Terraform, CloudFormation)
- Docker, Kubernetes
- CI/CD (GitLab, Jenkins, GitHub Actions)
- Monitoramento (Prometheus, Grafana, Datadog)
- Cloud (AWS, Azure, GCP)

## Qualidade

### Engenheiro de QA
- Estratégia de testes (unitário, integração, sistema, aceitação)
- Automação (Selenium, Cypress, Playwright)
- Testes de carga (JMeter, K6)
- Cultura de qualidade

## Design/UX

### UX Researcher
- Entrevistas
- Testes de usabilidade
- Análise de métricas
- Mapeamento de jornadas
- Criação de personas

### UX Designer
- Fluxos de navegação
- Wireframes
- Prototipagem interativa
- Arquitetura da informação

### UI Designer
- Design visual
- Identidade visual
- Design systems
- Tipografia, cores, iconografia
- Microinterações

### Design Ops
- Gestão de ferramentas
- Design System
- Processos de escala

## Produto

### Product Manager
- Descoberta de produto
- Roadmap
- Priorização (RICE, MoSCoW)
- Análise de concorrência
- OKRs

### Product Owner
- Gestão de backlog
- Histórias de usuário
- Aceite de entregas
- Interface com stakeholders

### Scrum Master
- Cerimônias ágeis
- Mentoria em práticas ágeis
- Melhoria contínua
- Proteção do time

## Estratégia

### Analista de Negócios (BA)
- Elicitação de requisitos
- Mapeamento de processos (BPMN)
- Documentação de regras de negócio
- Ponte cliente-técnico

### Customer Success Manager
- Relacionamento com cliente
- Satisfação
- Oportunidades de melhoria/expansão
- Gestão de contratos
