# Liderança Técnica - Equipe SBahia

## Responsabilidades

### Arquitetura MVC
- Definir estrutura de diretórios seguindo padrões MVC
- Estabelecer convenções de nomenclatura
- Definir padrões de comunicação entre Model-View-Controller

### Tomadas de Decisão
- Escolher stack tecnológica (Framework, DB, Cloud)
- Definir padrões de API (REST/GraphQL)
- Estabelecer guidelines de segurança

### Qualidade
- Code review e pair programming
- Definição de padrões de código (ESLint, Prettier)
- Revisão de arquitetura

## Competências Técnicas

### Essenciais
- Padrões de projeto (SOLID, DRY, KISS)
- Arquitetura de software escalável
- Segurança (OWASP Top 10)
- CI/CD e DevOps básico

### Conhecimento de Mercado
- Stack moderna (React/Next.js, Node/Express, PostgreSQL)
- Cloud (AWS/GCP/Azure)
- Containerização (Docker/Kubernetes)

## Workflow de Projeto

1. **Briefing** → Análise de requisitos
2. **Arquitetura** → Definição de estrutura MVC
3. **Planejamento** → Sprint planning com Tasks
4. **Execução** → Desenvolvimento agile
5. **Revisão** → Code review e QA
6. **Deploy** → Entrega contínua
