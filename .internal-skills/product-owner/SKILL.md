# Product Owner - Equipe SBahia

## Responsabilidades

### Gestão de Produto
- Priorização de funcionalidades
- Definição de MVP
- Roadmap do produto
- Métricas de sucesso

### Backlog
- User stories (formato INVEST)
- Critérios de aceite
- Refinamento contínuo
- Grooming com devs

### Validação
- Aceite de funcionalidades
- Demonstração ao cliente
- Feedback loop
- Ajustes de prioridade

## Competências Técnicas

### Frameworks
- Scrum / Kanban
- OKRs
- Metrics (North Star, Pirate Metrics)

### Priorização
- MoSCoW
- RICE
- Kano
- Value vs Effort

### Comunicação
- Stakeholder management
- Apresentação executiva
- Documentação clara
- Translation tech-business

## User Story Template

```
Como [persona],
Eu quero [ação],
Para [benefício].

Critérios de Aceite:
- [ ] Critério 1
- [ ] Critério 2

Definition of Done:
- [ ] Código revisado
- [ ] Testes passando
- [ ] Deployado em staging
```

## Métricas de Produto

- Activation rate
- Retention rate
- Revenue per user
- Net Promoter Score
- Time to value
